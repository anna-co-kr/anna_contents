## 0. 임원 요약 (한 문단)

Reel Studio는 **도구 자체가 목적이 아니라**, 안나의 글로벌 디자인 큐레이터 채널을 구축하기 위한 인프라다. 브리프에 기술된 2주 M1-M6 자동화 스코프는 "AI 생성 숏폼 공장형 채널"을 전제로 설계된 것이지, 확정된 포맷 — **"채널 IP가 글로벌 인디 브랜드 스토리를 전달, 인터뷰는 리서치 방법"** — 에 맞지 않는다. 따라서 브리프의 M1-M6은 폐기하고, 2주 MVP의 1순위 산출물을 **"도구"가 아니라 "@havea.official 계정에 출시된 영상 1편 (Birdie 커버)"** 으로 재정의한다. 도구는 그 1편을 돕는 선까지만 만든다.

---

## 1. 문제 정의

### 1.1 진짜 문제

안나는 Havea 플랫폼·유통 사업에서 **콘텐츠 중심 크리에이터**로 피벗하는 중이다. 진짜로 풀어야 할 문제는 두 층이다.

1. **상류 (아직 검증 안 됨)**: 안나의 감도·큐레이션·브랜드 서사 능력이 **글로벌 (EU/NA 디자인 큐레이터) 오디언스**에게 닿는가?
2. **하류 (도구가 다룰 영역)**: 솔로·영상 경험 거의 없음·편집 제약 (캡컷만 가능) 상태에서 어떻게 **지속 가능한 제작 리듬**을 만드는가?

### 1.2 브리프가 잘못 프레이밍한 문제

브리프는 이 문제를 **"영상을 빠르게 많이 만들기 위한 리소스 효율화"**로 프레이밍했다. 이건 틀렸다. 이유:

- 안나는 영상을 한 편도 만든 적이 없다 → 자동화할 대상이 뭔지 모른다
- 글로벌 수요는 0건 검증 → 효율화된 생산은 검증 안 된 콘텐츠를 더 빨리 만드는 것에 불과
- "솔로라서 자동화가 필요하다"는 운영 제약이지 **제품 요구사항이 아님**

올바른 프레이밍은 "**도구 없이도 영상 1편은 나와야 한다. 도구는 그 1편의 반응을 본 뒤에 무엇이 병목인지 아는 상태에서 만든다.**"

## 2. 수요 증거 (Demand Evidence)

### 2.1 구조적 사실

| 자산 | 숫자 | 해석 |
|---|---|---|
| @havea.official 팔로워 | 30K (활성 10K) | **"오디언스를 만들 수 있다"는 역량은 증명됨.** 전량 한국 타겟. |
| @birdie.korea 팔로워 | 168 (2개월, 인플루언서 시딩) | 느리지만 살아있음. 유의미한 사이즈 아님. |
| 글로벌 오디언스 반응 | 0건 | **실패가 아니라 null 신호.** 타겟을 잡은 적 없음. |
| Havea 리테일 채널 수 | 14개 | 창업자·브랜드 네트워크 = 인터뷰 다큐 장르의 필수 자산 |
| Birdie 수입 권리 | 확보 | **첫 영상의 콘텐츠 재료 확보됨** |

### 2.2 수요 가설의 현재 상태

- **한국 오디언스 빌딩 역량**: ✓ 증명됨 (30K Havea)
- **글로벌 오디언스에서 안나의 감도가 작동하는가**: ✗ 완전 미검증
- **"서울 기반 글로벌 인디 브랜드 큐레이터" 포지셔닝이 글로벌에 먹히는가**: ✗ 완전 미검증
- **10K 한국 활성 팔로워가 글로벌 피벗 과정에서 이탈·유지 여부**: ✗ 미검증 (리스크)

### 2.3 수요 증거의 부재가 설계에 주는 제약

V1은 **"검증되지 않은 가설을 가장 적은 비용으로 시험하는 장치"**다. 따라서:
- 크고 야심찬 콘텐츠 시리즈 아님
- 자동화 파이프라인 구축 아님
- **한 편의 영상 + 그 반응을 관찰할 수 있는 최소 장비**

안나가 V1 전에 실제로 돌리고 있는 워크플로우:

- **툴 스택**: 캡컷 (기초), Midjourney, Nano Banana, Higgsfield (구독 중). AE·Premiere·정식 Photoshop 없음.
- **제작 경험**: 이미지 콘텐츠 위주. 영상 거의 처음.

- **강점 영역**: 스토리텔링, 큐레이션 판단, 브랜드 서사, 감도 레퍼런스 (Indieshop·Simpsons·Scottie Cameron)
- **대화형 AI 작업은 재미있어함** — 이게 도구 UX의 제약 조건임 (폼 기반 UX 안 됨)

## 4. 타겟 유저 & 가장 좁은 웨지

### 4.1 채널 포지셔닝 (브리프에서 수정)

- **브리프의 포지셔닝**: "한국 브랜드를 글로벌에 소개하는 큐레이션 크리에이터"
- **수정된 포지셔닝**: **"서울 기반의 글로벌 인디 브랜드 큐레이터 (Seoul-lens on global design)"**
- **수정 이유**: Birdie 자체가 덴마크 브랜드이고 안나의 스코프는 다국가 인디 브랜드. "한국 브랜드 수출" 프레임은 실제 스코프와 어긋남. "서울에서 보는 글로벌 큐레이션"이 오히려 신선한 각도이며, 안나의 "한국인이라 글로벌 감도 캐치 약함" 자기 진단 리스크도 줄여줌 (큐레이터 역할이 감도 수신 쪽이므로).

### 4.2 타겟 오디언스 (W2)

- **1차 타겟**: EU/NA 25-40세, 독립 디자인·공예·리빙 관심, @kinfolk·@ignant·@cereal_magazine·@freundevonfreunden 인근 팬층
- **2차 타겟 (부차적)**: 북유럽·유럽 CO₂·실내공기·지속가능성 관심층 (Birdie 경로로 유입 가능)
- **3차 (명시적 비타겟)**: 한류 Gen Z. 포맷·감도 미스매치.

### 4.3 구체 오디언스 리서치

- **현재 상태**: 제로. 안나 본인이 "한 번도 해본 적 없음" 명시
- **V1의 부차 목표**: V1 출시 후 반응 데이터로 실제 오디언스 프로필 역산 (국가·계정·반응 유형)

### 4.4 가장 좁은 웨지 (Narrowest Wedge)

**@havea.official 계정에 Birdie 커버 영상 1편 (60-90초).** V1 전체 스코프가 이 한 줄에 담긴다.

- **왜 가장 좁은가**: 1 계정 × 1 영상 × 1 브랜드
- **왜 Birdie인가**: (a) 안나가 수입 권리 보유 = 100% 안나가 통제 가능, (b) 덴마크 태생 → 글로벌 도달의 자체 훅 ("Danish product thriving in Seoul"), (c) 기존 @havea.official 팔로워에게 친숙 앵커 → 갑작스러운 포맷 전환의 충격 완화
- **왜 @havea.official (신규 계정 아님)**: 기존 10K 활성 팔로워 = 0에서 시작하지 않음. 단, 기존 팔로워가 이탈할 리스크는 V1이 측정해야 할 지표 중 하나.

## 5. 제약 조건

### 5.1 자원 제약

- **인력**: 솔로 (안나 1인)
- **시간**: 2주
- **예산**: 기존 구독 외 신규 비용 최소
- **편집 툴**: 캡컷만 (AE·Premiere 사용 불가)

### 5.2 기술 제약

- **캐릭터 일관성**: AI 이미지 생성 툴의 character consistency는 불완전. Nano Banana character mode나 Midjourney `--cref` 사용 시 의상·조명·세부 디테일 컷마다 흔들림. LoRA 학습은 시간·비용 추가.
- **한국어 네이티브**: 안나는 영어 네이티브 아님. 영어 오디오 제작은 전달력 저하 리스크.

### 5.3 포맷·운영 제약

- **최종 영상 포맷 = Y** (채널 IP가 스토리 전달, 창업자 목소리·얼굴 등장 안 함)
- **계정 = @havea.official** (기존 30K, 한국 중심). 신규 채널 개설 아님.
- **오디오 전략 (V1 확정)**: **한국어 오디오 최소 + 영어 자막 메인 + 한국어 자막 서브**. 내레이션은 타이포그래피와 BGM으로 상당 부분 대체. 인터뷰 다큐 장르 문법에 부합.

### 5.4 철학 제약

- **스토리텔링·큐레이션 판단은 절대 자동화 안 함** — 안나의 강점. 도구는 속도만 증폭
- **대화성 보존** — 안나가 재미있어하는 AI 대화형 기획 경험 유지
- **IP는 결과물이 아닌 전제 조건으로 승격** — 브리프 7.3 수정. Y 포맷이면 IP가 있어야 첫 영상이 찍힘.

## 7. 검토된 접근법

### 7.1 Approach A — 수작업 1편, 도구 0

- 기존 툴 스택만으로 2주 안에 영상 1편
- **Completeness**: 5/10
- 장점: 확실한 완주, 최소 리스크
- 단점: Reel Studio 프로젝트 사실상 취소, gstack 학습 기회 상실, 2편째 다시 수작업

### 7.2 Approach B — 최소 인프라 + 인터뷰 기반 영상 1편

- Next.js 앱 (3모듈: transcript 마이닝, 레퍼런스 라이브러리, 캐릭터 락커) + 영상 1편 동시 제작
- **Completeness**: 7/10
- 장점: 완성되면 최고, 인프라 재사용
- 단점: 솔로·첫 영상에 도구 제작 + 인터뷰 섭외·촬영 + 편집 동시 진행 = 실패 확률 높음

### 7.3 Approach C — 캐릭터 탄생 오리진 필름이 첫 영상

- 첫 영상을 순수 IP 오리진 필름으로. Indieshop·Jacquemus 패턴
- **Completeness**: 9/10
- 장점: IP 디자인이 콘텐츠가 됨, 인터뷰 성사 리스크 제거
- 단점: **@havea.official 기존 계정에서는 브리지 없음 → 기존 팔로워 혼란**

### 7.4 **Approach C2 — Birdie가 곧 오리진 (최종 채택)**

C의 구조적 장점 + @havea.official 기존 계정의 현실을 통합. 첫 영상은 Birdie 소개인데, 그 안에 채널 IP의 오리진이 녹아 있음.

- **Completeness**: 9/10
- **Effort**: Human 2-3주 / CC+gstack 10-14일
- **Risk**: Medium

## 15. office-hours가 남긴 노트

세션 동안 안나의 사고 방식에서 관찰된 것들. 판단이 아닌 관찰.

- **"없어요"를 바로 말함** — Q1 "글로벌 수요 증거 있어요?"에 대해 우회 없이 "아직 시도하는 시점이라 증거 없음"으로 답. 많은 창업자는 이 질문에 "관심 있어하는 사람 많아요"로 얼버무림. 안나는 아님. 이게 데이터 기반 판단의 출발점이 된다.
- **"영상을 빨리 많이 만들고 싶다"와 "영상을 한 편도 안 만들어봤다"의 긴장을 본인이 감지 못함** — 초기엔 자동화가 필요하다는 가정이 너무 빨리 도착. 리프레이밍 제안에 저항 없이 수용한 건 좋은 신호. 다만 다음 세션부터 이 긴장을 본인이 먼저 감지하도록 훈련 필요.
- **제작 포맷을 오디언스보다 먼저 결정함** — "한국어 오디오 + 영어 자막"이 오디언스 리서치 없이 정해져 있었음. 이 패턴은 솔로 창작자에게 흔함. 다음 콘텐츠 결정에서는 **오디언스 → 포맷** 순서를 의식적으로 유지해야 함.
- **스토리텔링을 강점으로 명확히 자각** — "영상 만들 때 가장 공들이는 건 스토리텔링"이라고 본인 입으로 말한 것. 이게 도구 설계에서 "자동화 금지 영역"을 명확히 하는 근거가 됨. 드문 자기 인식.
- **감도 레퍼런스가 구체적이고 일관적임** — Indieshop·Simpsons·Scottie Cameron 셋이 서로 맞물림 (콜라주·시트콤 비트·유머러스 부조리). 이건 "좋아하는 것 많음"이 아니라 **취향의 축이 있다**는 증거. Taste Graph의 씨앗으로 유효.
- **브리프의 논리 오류 (IP가 결과물 vs IP가 전제)를 본인이 놓쳤지만, 지적 후 즉시 수용** — 방어하지 않음. 자기 문서에 대한 과도한 애착 없음. 빠른 반복을 가능하게 하는 성격.
