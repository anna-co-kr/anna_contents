## 1차 지표 — self_rating

```sql
-- 본 작업 세션의 prompts 평균 (session_id 기준 또는 created_at 범위)
SELECT 
  tool, language,
  COUNT(*) AS n,
  ROUND(AVG(self_rating)::numeric, 2) AS avg_rating
FROM public.prompts
WHERE user_id = '5aeaac31-9793-45fa-b276-d5f71304a530'
  AND created_at BETWEEN '<시작>' AND '<종료>'
  AND self_rating IS NOT NULL
GROUP BY tool, language;
```

```sql
SELECT 
  COUNT(*) AS pair_count,
  MAX(iteration_count_cumulative) AS max_iter,
  ROUND(AVG(iteration_count_cumulative)::numeric, 2) AS avg_iter,
  COUNT(*) FILTER (WHERE is_final_pick) AS final_picks
FROM public.pairs
WHERE user_id = '5aeaac31-9793-45fa-b276-d5f71304a530'
  AND created_at BETWEEN '<시작>' AND '<종료>';
```
